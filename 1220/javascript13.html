01자바스크립트-함수(function)

함수(function)이란?
함수는 일련의 내용을 삽입하여 사용자가 언제든 호출하여 내용을 실행할 수 있는 실행문
코드의 중복제거 및 재사용이 용이하다.
유지보수의 용이성이 높다.
함수의 종류는 내장함수(alert, console, prompt 등), 사용자정의 함수가 있다.
    1. 매개변수가 없는 일반 사용자 정의 함수 
    function 함수명(){ 
        실행할 내용; 
    } 

    2. 매개변수가 있는 함수 
    function 함수명(변수1, 변수2, 변수3, 변수4....){ 
        실행할 내용; 
    } 

    3. 리턴 값이 있는 함수  
    function 함수명(변수1, 변수2, 변수3, 변수4....){ 
        실행할 내용; 
        return 실행결과; 
    }

자바스크립트 함수 호이스팅

자바스크립트가 실행되면 parser가 빌드하면서 문법을 검사합니다. 이 와중에 끌어올릴 값을 선택하여 내부적으로 최상단에서 끌어 올려 처리합니다. 이것을 호이스팅 문법이라고 합니다.
